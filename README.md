# Monofile 

Monofile is a command line tool written in go that helps manage multi language mono repo. 

## Commands 
### Available 
This is work in progress and there are no commands available as of now.
### Planned 
- **init**: Initializes a new monofile project
- **add**: Adds a package to a project within mono repo
- **build**: Incrementally builds the projects 
- **test**: Tests all the projects (or selected)
- **changelog**: Allows to manage the changelogs for every git commits/push
- **dep-check**: Makes sure all the dependencies in the monorepo are of same version
